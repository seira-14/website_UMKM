# Website Support UMKM "Warung Nusantara"

## Deskripsi
Website Support UMKM "Warung Nusantara" merupakan website landing page yang dirancang untuk membantu pelaku UMKM mempromosikan produk kuliner khas Indonesia secara digital. Website ini menampilkan informasi usaha, daftar menu unggulan, testimoni pelanggan, serta halaman kontak yang memudahkan calon pelanggan untuk berinteraksi dengan pemilik usaha. Dengan tampilan yang responsif dan modern, website ini dapat diakses melalui berbagai perangkat, baik desktop maupun mobile.

## Bahasa Pemrograman
- HTML5
- CSS3
- JavaScript

## Fitur
- 🏠 **Hero Section** sebagai halaman utama dengan banner dan tombol Call-to-Action.
- 📖 **Tentang Kami** yang menjelaskan profil dan visi Warung Nusantara.
- 🍽️ **Menu Unggulan** yang menampilkan daftar makanan lengkap dengan gambar, nama menu, deskripsi, harga, dan tombol "Pesan Sekarang".
- ⭐ **Testimoni Pelanggan** untuk meningkatkan kepercayaan pengunjung.
- 📞 **Halaman Kontak** yang berisi:
  - Alamat usaha
  - Nomor telepon
  - Email
  - Jam operasional
  - Formulir pengiriman pesan
- 🌐 **Social Media** berupa ikon media sosial (Instagram, WhatsApp, Twitter, dan lainnya).
- 🧭 **Navigation Bar** yang memudahkan pengguna berpindah ke setiap section website.
- 📱 **Responsive Design** sehingga tampilan tetap optimal di berbagai ukuran layar.
- 🎨 **Modern User Interface (UI)** dengan kombinasi warna yang menarik dan tata letak yang rapi.

## Cara Menjalankan
1. Download atau clone repository ini.
2. Buka folder project.
3. Jalankan file `website_umkm.html` menggunakan browser.
4. Website siap digunakan.

## Tujuan Project
Project ini dibuat sebagai media promosi digital bagi UMKM agar dapat memperkenalkan produk kepada masyarakat secara lebih luas melalui website yang informatif, menarik, dan mudah digunakan.
