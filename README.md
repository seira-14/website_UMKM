# Website Support UMKM "Warung Nusantara"

## Deskripsi
Website Support UMKM "Warung Nusantara" merupakan website landing page yang dirancang untuk membantu pelaku UMKM mempromosikan produk kuliner khas Indonesia secara digital. Website ini menampilkan informasi usaha, daftar menu unggulan, testimoni pelanggan, serta halaman kontak yang memudahkan calon pelanggan untuk berinteraksi dengan pemilik usaha.

## Bahasa Pemrograman
- HTML

## Fitur
- Hero Section sebagai halaman utama dengan banner dan tombol Call-to-Action.
- Tentang Kami yang menjelaskan profil dan visi Warung Nusantara.
- Menu Unggulan yang menampilkan daftar makanan lengkap dengan gambar, nama menu, deskripsi, harga, dan tombol "Pesan Sekarang".
- Testimoni Pelanggan untuk meningkatkan kepercayaan pengunjung.
- Halaman Kontak yang berisi:
  - Alamat usaha
  - Nomor telepon
  - Email
  - Jam operasional
  - Formulir pengiriman pesan
- Social Media berupa ikon Instagram, WhatsApp, dan Twitter.
- Navigation Bar yang memudahkan pengguna berpindah ke setiap section website.
- Responsive Design sehingga tampilan tetap nyaman digunakan pada berbagai ukuran layar.
- User Interface (UI) yang sederhana, modern, dan mudah digunakan.

1. Download file `website_umkm.html` .
2. ketik ctrl + o di browser.
3. pilih file `website_umkm.html` .
4. Website siap digunakan.

## Tujuan Project
Project ini dibuat sebagai media promosi digital bagi UMKM agar dapat memperkenalkan produk kepada masyarakat secara lebih luas melalui website yang informatif, menarik, dan mudah digunakan.
